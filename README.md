# Minsait
Proyecto de formacion en empresa de Minsait que trata sobre una web de eventos sostenibles

## Tecnologias usadas
HTML, CSS
C#
SQL
Visual Studio Code
Virtual Box
XAMPP
Git + Github

## Funcionalidades en el programa
Listado de eventos
Registro y gestion de usuarios
Gestion de organizadores y sus eventos
Inscripcion y cancelacion a eventos
Tipos de evento
Ubicaciones
